# Python-Study-Log 🐍📘
파이썬 문제 해결 과정과 사고 과정을 기록하기 위한 저장소입니다  
Pythonの問題解決プロセスと思考の流れを記録するためのリポジトリです 

## 🎯 학습목표 学習目標 🎯

- 문제를 완전히 해결하기 전까지는 정답을 보지 않기
- 問題を完全に解決するまでは、答えを見ないこと 🔍

  
  
- 문제를 해결하며 배운 사고과정 기록 및 새롭게 배운 문법 기록
- 問題を解きながら得た思考プロセスや、新しく学んだ文法を記録する 🧠✏️

  
  
- 문제 해결 후, 더 효율적인 코드를 찾기 위하여 chat GPT를 통해 코드를 개선하고 기록
- 問題解決後、より効率的なコードを書くためにChatGPTを活用し、コードを改善・記録する 🤖💡



## 📂 구성요소 構成要素 📂

- **🧩 problems 🧩**  
  각 문제별로 하나의 폴더를 생성하며, 각 폴더에는 다음 파일들이 포함됩니다
  各問題ごとに1つのフォルダを作成し、以下のファイルを含めます
  
  - `solution.py`: 나의 초기 풀이  自分の初期解法 💻
  - `improved.py`: 더 나은 풀이, 리팩토링 버전  より良い解法（リファクタリング版） 🚀 
  - `README.md`: 문제를 풀며 배운 점 기록  問題を通して学んだことの記録 📝

- **📒 notes 📒**  
  알고리즘, 사고과정, 프로그래밍에 필요한 로직을 정리
  アルゴリズム、思考プロセス、プログラミングに必要なロジックを整理 📚

- **🛠 utils 🛠**  
  여러 문제에서 자주 사용하는 헬퍼 함수나 공통 라이브러리 코드를 기록
  複数の問題でよく使うヘルパー関数や共通ライブラリコードを記録 🔧
